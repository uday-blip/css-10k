Combinators

Descendent selectors

These selectors are used to select all elements and apply elements of the block level element (whole div tag) , this descendent selector is indicated by using space ( )

Child selectors

the child selectors are used to select only the child elements of the container or div tag that we have took in our css , this child selector is indicated by using greater than symbol (>)

Adjacent sibling selectors

The Adjacent sibling selector are used to select and apply styles to the block level element which is very next to the element that we took in our css, This Adjacent sibling selector is indicated by using the plus symbol (+)

General Sibling Selector

The General Sibling selector is used to select and apply styles to all the child elements that are after to the block level element to which we took in our css, The general sibling selector is indicated by using the symbol tilt (~)
